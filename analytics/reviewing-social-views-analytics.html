    소셜 뷰 분석 검토

    이 항목에서는 브라이트코브 소셜을 사용하여 동영상의 소셜 조회수 성능을 모니터링하는 방법에 대해 알아봅니다.

    브라이트코브 소셜은 소셜 미디어 계정에 게시된 비디오의 성능을 모니터링하는 데 사용할 수 있는 여러 보고서를 제공합니다.

    • 대시보드 - 소셜 미디어 계정의 비디오 성능에 대한 높은 수준의 개요를 제공합니다. 자세한 내용은소셜 분석 검토를참조하십시오.
    • 소셜 보기 - 소셜 미디어 계정에 공유되는 비디오의 소셜 보기의 개요를 제공합니다. 이 항목에서 다룹니다.
    • 소셜 참여 - 소셜 미디어 계정에 공유된 동영상의 소셜 참여 (좋아요, 공유, 리트윗 등) 에 대한 개요를 제공합니다. 자세한 내용은소셜 참여 분석 검토를참조하십시오.

    보고서의시간 범위를 선택하여시작합니다. 소셜 뷰 분석 보고서를 보려면 페이지 상단에 있는소셜 뷰탭을 클릭합니다.

    소셜 뷰 분석 보고서는 다음 섹션으로 구성됩니다.

    • 소셜 뷰
    • 소셜 뷰 세부 정보

    소셜 뷰

    소셜 조회수 그래프에는 선택한 기간 동안 소셜 미디어 계정에 게시된 비디오의 비디오 조회수 수가 표시됩니다. 업데이트 날짜는 Video Cloud에 소셜 분석 데이터가 있는 최신 날짜입니다. 일반적으로 소셜 플랫폼이 API의 뷰 및 참여 데이터를 업데이트하기 전에 24-48시간 이상 지연이 발생합니다.

    참고: 트위터 인사이트 API는 미디어 보기를 보고하지 않으므로 소셜 분석의 일부로 데이터 보기를 사용할 수 없습니다.

    그래프 오른쪽에 있는 범례를 사용하여 표시할 세부 수준을 선택할 수 있습니다.

    • 내 사이트 및 앱 - Video Cloud 플레이어 및 앱의 비디오 조회수 표시
    • 페이스 북 - 페이스 북에비디오 보기의 수를 표시
    • 유튜브 - 유튜브에비디오 보기의 수를 표시
    • 총 - 모든 플랫폼에서 총 비디오 시청 횟수 표시

    그래프의 선 위에 마우스를 놓으면 해당 날짜의 데이터가 표시됩니다.

    소셜 뷰 세부 정보

    소셜 뷰 세부 정보 섹션은 다음을 통해 비디오 보기 세부 정보를 볼 수 있습니다.

    • 뷰 (기본값)
    • 평균 시청 시간
    • 평균 감시률 (%)

    선택한 후 보고서에는 각 비디오에 대한 자세한 데이터가 표시됩니다.

    참고: 트위터 인사이트 API는 미디어 보기를 보고하지 않으므로 소셜 분석의 일부로 데이터 보기를 사용할 수 없습니다.

    분석 보고서에 표시된 YouTube 조회수는YouTube 분석 개요보고서에서 수집됩니다. YouTube는 이 데이터가 실시간 데이터임을 보장하지 않습니다. 따라서 이 보고서에 보기 및 참여가 나타나기 전에 며칠의 지연이 나타날 수 있습니다. 데이터가 YouTube 분석에 저장되면 브라이트코브 소셜 분석을 위한 야간 데이터 수집가가 선택합니다.

    최신 실시간 시청 데이터가 필요한 경우 YouTube는 가장 최근에 게시된 상위 25개 동영상에 대한 지난 48시간 동안의 시청 데이터 추정치와 함께실시간 보고서를제공합니다. YouTube 채널에 대한 실시간 보고서는여기에서확인할 수 있습니다. 브라이트코브 소셜 애널리틱스는 실제 조회수와 달리 예상 조회수를 제공하고 최근 동영상에만 적용되므로 이러한 실시간 데이터를 수집하지 않습니다.

    보고서 하단에는 표시할 행 수를 선택하고 다른 페이지로 이동할 수 있는 컨트롤이 있습니다. 동영상 제목에 대한 링크를 클릭하면 선택한 동영상의 소셜 조회수 분석이 표시됩니다.

    일반적인 질문

    1. Social 모듈을 사용하여 YouTube에 동영상을 게시한 다음 해당 동영상이 다른 사이트의 YouTube 플레이어에 삽입되면 재생/참여 데이터가 수집되어 소셜 모듈 분석에 전달됩니까?

    소셜 분석은 크리에이터 스튜디오 및/또는 YouTube의 분석 페이지에 표시되는 대로 YouTube에서 보고한 모든 조회 데이터를 수집합니다. YouTube 내장 플레이어 조회수가 분석에 반영되면 Social 모듈에서 이를 캡처하여 보고합니다. 임베드 코드 YouTube의공유 > 임베딩기능을 지정된 대로 사용하면 포함된 플레이어의 조회수가 YouTube의 분석에 반영됩니다. 하지만 임베딩 코드를 편집하면 YouTube에서 조회수 데이터를 수집하지 못할 수 있으며, 특히 포함된 플레이어가 동영상을 자동으로 재생하도록 설정하면 YouTube가 해당 플레이어의 조회수를 계산하지 못하는 것으로 보입니다.

    데이터 내보내기

    보고서 데이터를 쉼표로 구분된 값 및 Excel 형식 양쪽 모두로 다운로드할 수 있습니다. 내보내기 기능은Analytics API호출에서 직접 데이터를 저장합니다. 내보낸 데이터는 UI 데이터와 같은 방식으로 조작되지 않으며, UI에 표시되는 필드 하위 집합만이 아니라 현재 보고서에 포함된 모든 기본 데이터를 전부 포함하고 있습니다. API의 원래 형식으로 모든 데이터를 보유하면 UI에 제공되는 것 이외의 방식으로 데이터를 조작 할 수 있습니다.
